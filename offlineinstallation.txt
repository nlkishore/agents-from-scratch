Step 1: On the Internet-Connected Machine
1.1. Create a virtual environment (optional but recommended)
python -m venv langgraph_env
source langgraph_env/bin/activate  # On Windows: langgraph_env\Scripts\activate

î·™î·š
1.2. Create a requirements.txt file
Include LangGraph and any other dependencies you need:
langgraph-cli[inmem]
langchain
langchain-openai
python-dotenv


1.3. Download all required packages
mkdir langgraph_packages
pip download -r requirements.txt -d langgraph_packages


This will download .whl and .tar.gz files for LangGraph and all its dependencies into the langgraph_packages folder.
1.4. (Optional) Freeze exact versions for reproducibility
pip freeze > exact_requirements.txt


1.5. Transfer the folder
Copy the langgraph_packages folder and requirements.txt (or exact_requirements.txt) to the offline machine using USB, shared drive, etc.
